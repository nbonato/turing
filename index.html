Press and politics in the UK

Slide the indicator on the left to select the year
Use the buttons on the right to choose the dataset to show


Click on any available county to see the data. If a county is grayed out, it means that there are no press information on it for that year, so try moving around the slider above.

This project was created by Nicolò Bonato for a digital residency sponsored by The Alan Turing Institute.

The press data comes from the Press Directories dataset created by the Living with Machines project.

The electoral data comes from the CLEA project at the University of Michigan. This project made use of geographic data provided by the Historic County Borders Project.

All code is available on GitHub on the Visualising Press Directories repository.
